# Creative Bible Exposition

A single-page website for **Creative Bible Exposition** — a ministry that equips ordinary preachers with simple, powerful Bible study methods using both sides of the brain.

## About

Creative Bible Exposition is shaped for in-depth study and clear exposition. Each study begins with the Biblical text, follows the background, and turns it into a relatable sermon. Founded by Dr. Yap Beng Shin, the goal is to help preachers who have little access to resources or higher education.

## Tech Stack

- **HTML5** — Semantic markup with full accessibility support
- **CSS3** — Custom properties, animations, glassmorphism, responsive design
- **Vanilla JavaScript** — Scroll-reveal animations, mobile navigation, form handling
- **Google Fonts** — Inter (body) + Libre Baskerville (serif accents)
- **Custom Font** — Brolian (headings)
